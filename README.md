# Arduino_Mega_Programmer

1) To read ROMs, EPROMs, Flash of older Microcontrollers or Computers
2) Read memory and save BIN or TXT format file
3) Using Arduino Code and Golang (only) 
4) OS = Windows 10 (for while)
